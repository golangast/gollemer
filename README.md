# gollemer: LLM Command-line Utility

This project provides a command-line utility for interacting with Large Language Model (LLM) functionalities.

## Usage

To run the LLM utility, use the following command from the project root:

```bash
go run . -llm "Your query here"
```

Replace `"Your query here"` with the natural language input you want the LLM to process.

## Example

```bash
go run . -llm "What is the capital of France?"
```
This will execute the LLM functionality with the provided query.
